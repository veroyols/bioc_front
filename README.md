# 🌐 Bioconductor Frontend
Este repositorio contiene el código fuente para la interfaz web del proyecto Bioconductor. El frontend está desarrollado en React y sirve como visualizador e interfaz interactiva para explorar y analizar datos genómicos procesados en el backend.

## 📦 Características
* Visualización Genómica: Representación gráfica de cromosomas, genes y alineamientos basados en datos genómicos.
* Interactividad: Permite explorar datos cargados, incluyendo GFF3, BAM/SAM y archivos FASTQ.
* Integración con Backend: Comunicación fluida con el backend en R mediante APIs RESTful.
* Filtros y Búsquedas: Opciones para filtrar regiones genómicas o genes específicos.
* Amigable para Usuarios: Diseñado para facilitar el uso por parte de bioinformáticos y otros científicos.
